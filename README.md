# Glow-Up Advisor — Your AI Skincare Companion

**About**

Glow-Up Advisor is an AI-powered skincare companion that helps you build personalized routines, track progress, and get expert advice.

**Getting Started**

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Create an `.env.local` file and set the right environment variables

```
VITE_API_URL=http://localhost:5001
```

Run the app: `npm run dev`

**Backend**

The backend lives in `GlowupAi-backend/`. See that folder for setup instructions.
